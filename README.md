# Cart shop Server

An e-commerce backend server test

- please view on large screen. Pleasant looking UI not intented only important functionalities.

## Project Stack

- NodeJs
- MongoDb
- Express
- Ejs Template Engine

## Installation and Setup Instructions

Installation:

`npm install`  

To Run Test Suite:  

`npm test`  

To Start Server:

`npm start`  

To Visit App:

`localhost:5000`
